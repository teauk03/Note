# Array
![alt text](<../설명사진/[자료구조-선형구조] array01.png>)
* **Array(배열)이란** 동일한 데이터의 타입을 가진 요소들을 연속적인 메모리 공간에 일렬로 저장하는 기본적인 선형 자료구조
* 데이터를 순서대로 나열하며, 각 데이터에는 `인덱스(index)`라는 고유한 번호를 가짐
    * 이 index는 0부터 시작함

## Array의 구조
1. **길이가 고정 되어 있음**
    * 한번 인덱스의 길이를 선언하면 그 길이가 고정됨
2. **확장에 불리함**
    * 데이터가 선언된 인덱스의 길이보다 많아진다면, 
        1. 더 큰 배열을 다시 생성하고
        2. 기존에 저장한 데이터를 복사하고
        3. 저장된 데이터를 더 큰 배열에 붙여넣는 짓을 해야함
3. 메모리 공간이 낭비될 수 있는 구조
    * 배열을 많이 만들어놓고 그 배열을 사용하지 않으면 메모리 낭비로 이어질 수 있음

# 시간 복잡도 

## 읽기 or 접근 = O(1)
* 인덱스를 알고 있다면 수학 공식으로 바로 계산하면 나옴
* 가장 큰 장점임

## 검색 = O(N)
* 특정 값이 어디 있는지 찾을려면 `index[0] ~ [index N]` 까지 찾아봐야함 

## 중간에 데이터 제어(삽입/수정/삭제) O(N)
* index[0]에 있는 새로운 값을 제어할려면, 기존에 `index[1] ~ index[N]` 까지 모든 데이터를 한 칸씩 이동해야함
