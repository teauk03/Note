# Uzi's memory storage
* 이 리포지토리는 저의 학습 기록을 저장하는 공간입니다.
* 저는 제가 학습한 내용을 100% 기억하지 못하기에 이러한 저장소를 만들었습니다.
* 때문에 저의 철학은 `불확실하고 유한한 기억보단 무한하며 보다 확실한 기록`입니다.
* Note Repo에선 **핵심 개념 위주로 기록**하며, <U>기록된 내용을 읽고 빠르게 상기(想起)할 수 있게 하는 것</U>을 목표로 합니다.

<br>

# 폴더 구조
* **`computer_architecture`** : 컴퓨터 구조의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다.
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`db`** : 데이터베이스의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다.
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`oopd`** : 객체 지향 패러다임의 디자인 패턴에 대해 학습한 내용을 정리한 폴더입니다.
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`jpa`** : JPA의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다.
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`network`** : network의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다.
