# Uzi's memory storage
* 이 리포지토리에서는 저의 학습 기록을 저장하는 공간입니다. 
* 저는 저의 학습된 내용을 100% 기억하지 못하기에 이러한 저장소를 만들었습니다. 
* 때문에 저의 철학은 `불확실하고 유한한 기억보단 무하하며 보다 확실한 기록`을 입니다.
* Note Repo에선 핵심 계념 위주로 기록하며, 해당 기록된 내용을 읽고 빠르게 기억할 수 있게 하는 것을 목표로 합니다. 

<br></br>

# 폴더 구조
* **`computer_architecture`** : 컴퓨터 구조의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다. 
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`db`** : 데이터 베이스의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다. 
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`oopd`** : 객체 지향 패러다임에 대한 디자인 패턴에 대해 학습을 정리한 폴더입니다. 
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`jpa`** : JPA의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다.
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`network`** : network의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`os`** : 운영체제의 핵심 개념에 대해 학습한 내용을 정리한 폴더입니다. 
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`security_principles`** : 시스템을 개발하면서 지켜야하는 최소한의 보안 원칙을 정리한 폴더입니다. 
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
* **`spring_framework`** : Java 기반 프레임워크인 Spring 프레임워크를 학습한 내용을 정리한 폴더 입니다. 
    * `어노테이션`: 스프링 프레임 워크에서 제공하는 어노테이션에 대해 학습한 내용을 저장하는 폴더입니다. 
    * `설명사진` : 해당 챕터에 대해 직관적으로 이해하기 쉽게 제작한 사진이나 그림을 저장하는 폴더입니다.
